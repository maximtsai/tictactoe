Tic tac toe by Maxim Tsai
=====
To run this game, first start a local server in this folder. If you have python installed, you can click on runServer.bat. runserver.bat is pointing to port 1234 and can be accessed by going to localhost:8124 in a web browser.

This game can also be played on my website at:
maximtsai.com/games/tictactoe
